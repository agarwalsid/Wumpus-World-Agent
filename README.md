This project uses SWI Prolog’s ability to do inference in order to write an agent that will make safe moves within the Wumpus World.

Procedure To Run:
•	Open SWI Prolog and load the directory
•	Start SWI-Prolog: pl
•	Load the world simulator: ?- [wumpus_world].
•	Load the agent: ?- [my_agent].
•	Run the agent: ?- evaluate_agent(1, Score, Time)
•	Correct to: "wumpus:evaluate_agent(1, Score, Time)"? y
